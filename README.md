# KLE - Laravel Ürün Yönetim Sistemi

Bu proje, Docker (Laravel Sail) ortamında geliştirilmiş, kullanıcı yetkilendirmeli kurumsal bir Ürün CRUD uygulamasıdır.

## Kurulum ve Çalıştırma Adımları

1. Docker Desktop uygulamasını başlatın.
2. Proje kök dizininde terminali açıp bağımlılıkları yükleyin ve Sail'ı ayağa kaldırın:
```bash
   ./vendor/bin/sail up -d
